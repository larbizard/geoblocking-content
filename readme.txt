Geo blocking content and Broadcasting blackout.

== Description ==
Geo blocking content plugin allows you to set a content blackout for a specific city blocking users with IP address related to this region from accessing the content.

== Installation ==
1. Add plugin and Active as a classic plugin
2. Go to post or page and select the city to block in the list
3. Save you post
